# Build landing page

## Description

As a potential user, I want to visit a landing page that explains what Dagger is, how it works, and how to use the API so I can evaluate whether to adopt it.

## Acceptance criteria

1. Static HTML/CSS page hosted on Railway (or free static host like Netlify)
2. Explains: what Dagger is, the problem it solves, how agents use it
3. API reference section documenting:
   - POST /api/v1/publish
   - GET /api/v1/tickets/:displayId
   - GET /api/v1/documents/:id
   - Authentication (API keys)
4. Clean, minimal design — developer-focused
5. Links to GitHub repo and contact
